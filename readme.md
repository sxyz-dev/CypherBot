## 🚀 Tentang CypherBot

CypherBot adalah project bot WhatsApp yang dibangun menggunakan library [Axell Baileys](https://github.com/AxellNetwork/Baileys). Saat ini dalam tahap beta testing dengan pengembangan aktif!
